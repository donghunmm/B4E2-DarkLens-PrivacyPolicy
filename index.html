다크패턴 탐지 AI - 개인정보처리방침

최종 업데이트: 2025년 12월 15일

1. 수집하지 않는 정보

본 다크패턴 탐지 AI 크롬 확장 프로그램은 사용자의 개인정보(이름, 이메일, IP 주소, 브라우징 기록 등) 또는 민감한 데이터를 일체 수집, 저장, 전송하지 않습니다.

모든 분석은 사용자의 현재 웹페이지 DOM에서 로컬로만 수행되며, 외부 서버와 통신하지 않습니다.

2. 권한 사용 목적

확장 프로그램은 activeTab 또는 웹페이지 접근 권한만 사용해 다크패턴(예: Bait and Switch, Hidden Information)을 탐지합니다.

이는 페이지 콘텐츠 분석에 한정되며, 사용자 식별 정보는 생성되지 않습니다.

3. 데이터 저장

로컬 스토리지에는 사용자 설정(예: 탐지 민감도 등 일부 환경설정)만 저장되며, 이는 브라우저 내에서만 접근 가능합니다.

시크릿 모드에서도 동일한 정책이 적용되며, 서버로 전송되는 데이터는 없습니다.
